
# Setxo Components

React Components with Tailwind CSS , optimized for Laravel and Inertia JS

## Installation

```bash
  npm install
```
    
## Install Setxo Package

```bash
  npm install setxo-components
```
## Run Project

```bash
  npm run dev
```
## Usage/Examples

```javascript
import Button from 'setxo-components'

function App() {
  return <Button />
}
```
